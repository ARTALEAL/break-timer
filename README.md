# break-timer

Приложение таймер для организации перерывов во время собраний, обучения или встречь.

## Стэк

- HTML5
- CSS3
- JavaScript

## Варианты использования

- Можно использовать в браузере.
- Можно сохранить на экране "Домой" вашего мобильного устройства.
